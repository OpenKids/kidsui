kids-button

Click Me! Hello! Big Button Tiny Disabled

Attributes

NameTypeDefaultDescription
variant string "primary" Button style: primary, secondary, accent, warning
size string "md" Button size: sm, md, lg
disabled boolean false Disables the button interaction

Animations

  • Entrance: Pops in with spring scale
  • Hover: Gentle scale-up + shadow lift
  • Press: Squishes down with spring
  • Focus: Ring pulse

<kids-button disabled>Disabled</kids-button>

kids-timer

Start Timer Pause Reset

Attributes

NameTypeDefaultDescription
duration number 60 Total time in seconds
mode string "countdown" Timer mode: countdown, countup
running boolean false Starts/stops the timer
variant string "default" Style: default, compact, circular
warn-at number 10 Seconds remaining to show warning state

Methods

  • reset() — Resets the timer to initial state

Animations

  • Entrance: Spring pop
  • Warning: Pulsing red glow
  • Complete: Bounce
 
<!-- Countdown timer -->

<!-- Circular variant -->

<!-- Control timer -->
<script>
  const timer = document.querySelector('kids-timer');
  timer.setAttribute('running', '');   // Start
  timer.removeAttribute('running');   // Pause
  timer.reset();                      // Reset
</script>
